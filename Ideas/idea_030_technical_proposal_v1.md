## 0. 一句话

在 D-FINE 上,用**免监督零参数的频谱统计判据**(局部高频异常度)替换 Dome-DETR 的学习式密度头(DeFE, 0.8M + GT 高斯密度图 + DRFL 专用损失),驱动浅层 encoder token 的稀疏化与 query 预算分配——同通路、判据免费。

## 1. 问题定义与定位

- **通路已被 Dome-DETR 证明有效**(+2.5~3.3 AP):判据热图 → 二值掩码 → 浅层 token 稀疏 + query 自适应。#30 的科学问题不再是"这条路通不通",而是:**判据能不能免费?**
- **判据成本对照**:

| | Dome DeFE | #30 频谱判据 |
|---|---|---|
| 参数 | 0.8M 专用头 | **0**(统计量) |
| 监督 | GT 高斯密度图(核∝bbox) | **无**(免标注) |
| 损失 | DRFL(√d_gt 加权 MSE + 低估惩罚) | **无** |
| 跨数据集 | 判据头需重训 | **免调直接迁移**(至多一个阈值) |
| 判据算力 | conv 级联+通道注意力前向 | 深度卷积高通/池化统计(约 1/40) |

- **定位声明(取 Dome 教训)**:Dome GFLOPs +37%(154.2 vs 112.4),自认"加浅层特征换精度、稀疏化控成本"。#30 沿用同一诚实定位——**不是效率方法,是"以受控成本引入浅层高分辨率信息"的方法**;判据免费化是差异所在,叙事绝不许诺"净省算力"(除非实验证实)。

## 2. 判据设计(核心技术选项)

### 2.1 三选项(与 A轨 #11 共用判据族,EFSI 教训直接吸收)

| 选项 | 实现 | 依据 | 风险 |
|---|---|---|---|
| **S1 空域高通代理(首选)** | 固定核深度卷积(Laplacian / Sobel 幅值)→ 平方 → 局部异常度归一 | **EFSI 硬消融:空域代理 33.1 > FFT 32.3 > DWT 32.1**;kernel 可融合、无复数张量、边缘设备友好 | 固定核是否足够判别(vs 可学习高通)——留消融 |
| S2 DCT 块统计(备选) | 8×8 块 DCT → 高频系数能量占比 | FcaNet 证明 GAP=最低频 DCT,DCT 与通道统计天然衔接;JPEG 生态成熟 | 块效应;实现比 S1 重 |
| S3 FFT 频段能量(对照) | rFFT → 低/中/高三频段能量向量 | #11 v1.0 原方案;cuFFT 成熟 | EFSI 四条工程理由(kernel 融合难/复数带宽/形状敏感/NPU 无优化核);消融显示可能反而更差 |

**决策**:S1 为默认实现,S2/S3 为消融对照。叙事随之从"FFT 判据"抽象为"**高频响应统计判据(实现无关)**"——这同时规避与 FFT 系竞品(UAV-DETR/SET)的工具层撞车,把贡献锚在"判据→预算分配"的范式层。

### 2.2 判据公式(继承 #11 修正版,SET 警告已吸收)

```
输入: 最浅层特征 F ∈ (B, C, H, W)   # 与 Dome DeFE 同一接入位置,公平对照
1. 高频响应:   R = Σ_c |DWConv_hp(F)|²        # S1: 固定 Laplacian 核,逐通道后求和
2. 局部异常度: A = (R − μ_local) / (σ_local + ε)   # AvgPool 窗口 k=7
   # 点状孤立高值(小目标) → A↑;线状连续高值(建筑边缘) → A↓ —— 回应 SET"背景高频噪声"警告
3. 窗口聚合:   S_w = max-pool(A, window)       # 与 Dome MWAS 同窗口粒度(H/W=10 起步)
4. 预算分配:   前景窗 = {w : S_w > τ_adapt}     # 自适应阈值,保底机制照抄 Dome(T_init 递减至少激活一窗)
```

- **零参数版**为主打;**+1 参数版**(可学习 τ)与 **线性组合版**(w·[A_low,A_mid,A_high]−τ, 仅 4 参数)作为消融阶梯,证明"判据不需要 0.8M"
- 极小目标偏低频问题(SET):S1 主打版先不做三频段,由消融 F3(三频段异常度)验证是否必要——DETR 浅层 token 稀疏的容错好于 YOLO 门控(误保留只是多算几个窗,误丢弃才致命 → 阈值偏保守 + 保底机制兜底)

## 3. D-FINE 接入点设计

### 3.1 接入位置(与 Dome 逐点对齐,便于对照)

```
D-FINE(HGNetv2 + 1层 encoder AIFI + CCFF + 6层 decoder FDR)
│
├─ [接入点 P] backbone 最浅层特征(stride-4)
│    └─ 判据计算(§2)→ 窗口二值掩码 M
│
├─ [改造点 E] 浅层特征进 encoder 的方式:
│    仅前景窗 token 参与窗口注意力(结构沿用 Dome MWAS 的
│    「窗内 MSA → 轴置换跨窗 MSA → FFN」,该部分非本方案贡献,显式引用 Dome)
│    背景窗 token: 直通(identity)保留原值 —— SViT 教训:硬删除是崩溃根因
│
└─ [改造点 Q] query 初始化预算:
     Top-K 候选中,弹性部分用掩码 M 过滤(Dome PAQI 简化版:
     不引入 Dynamic NMS —— 保住 D-FINE 端到端性 = 对 Dome 攻击面③的直接回应)
```

### 3.2 与 Dome 的组件级差异表(写论文时的 Method 对照骨架)

| 组件 | Dome-DETR | #30 | 差异性质 |
|---|---|---|---|
| 判据 | DeFE(0.8M, GT 监督, DRFL) | 频谱统计(0 参数, 免监督) | **核心贡献** |
| token 稀疏 | MWAS | 沿用 MWAS 结构(引用) | 非贡献,控制变量 |
| query 预算 | PAQI + Dynamic NMS(3 超参) | 掩码过滤,**无 NMS** | 次贡献(端到端保持) |
| 背景 token | 掩码外不参与注意力 | 同 + 显式 identity 直通 | 工程细节 |

> 控制变量原则:token 稀疏结构照抄 Dome,把全部差异集中在判据与 NMS 两点——审稿人无法把增益归因到无关结构改动。

## 4. 对照叙事(Dome 三攻击面 → #30 三卖点)

| Dome 攻击面 | #30 回应 |
|---|---|
| ① GFLOPs +37% 非净省 | 判据部分成本 ≈1/40;总成本仍诚实定位为"受控引入浅层信息",但**判据免费使 S 档模型的额外开销显著低于 Dome-S** |
| ② 判据需 GT 密度监督+重训 | 免监督统计量,跨数据集(VisDrone↔AI-TOD)**零重训迁移**——独有实验:判据跨数据集直接换用 |
| ③ 保留 Dynamic NMS 破坏端到端 | 不引入 NMS,保持 D-FINE 端到端谱系纯洁性 |

**双轨验证卖点**:与 🟦 A轨 #11(YOLO P2 门控)同判据、跨架构 → ⬜ #24 信息瓶颈理论的"架构无关性"证据。

## 6. 风险与退守

1. **E1 输掉 >1 AP**:退守"成本-精度谱系"叙事(判据成本轴上的帕累托点),仍可发,档次降
2. **免监督判据在极密集场景失效**(AI-TOD 密集簇):保底机制+保守阈值兜底;E3 提前暴露
3. ~~**Dome accept 后放码**(🔔 已挂跟踪):放码反而利好~~ → **✅ 已发生(2026-07-18 检索确认)**:ACM MM 2025 接收+代码/权重全放出;E1 正式升级为官方对照,MWAS 结构照抄可落到代码级(⏸ 代码细读待实验模块);利好兑现
4. **撞车窗口**:频域 DETR 六→七篇/年的节奏(2026-07-18 FSDETR IJCNN 2026 = 第 7 篇,仍为增强范式),「频谱判据+条件计算」空白可能 6–12 个月内被填 → #30 排 B轨最高优先
5. **HF-DETR(IEEE SPL 2026,2026-07-17 检索命中)🔔**:「高频增强 + SSMG saliency token 门控 + VisDrone 实时」——迄今组合空间最近的邻居。现有信息判定:其高频组件做**特征增强**非判据统计,SSMG 大概率为**可学习微门**(落 learnable gating 已占区)→ 与 #30「免监督统计判据→预算分配」仍可划界;但 **SSMG 判据性质须全文核实**(无 arXiv,IEEE 拿不到;2026-07-18 复查:仍付费墙)——若为免监督统计门则直接命中核心卖点,须启动叙事调整。附带正面证据:其 FSD-Stem 用 LoG(Laplacian 家族)做高频提取 = S1 工具选择的又一独立佐证。详见 [HF-DETR 快评](../papers/summaries/HF-DETR_SPL2026.md)
6. **HI-MoE(arXiv 2026.04, preliminary)**:DETR 两级 MoE(scene→instance 路由),DINO +3.3 APs——条件计算在**专家维**非 token 空间维、判据为学习式路由 → 与 #30 正交;其"条件计算利好小目标"结论与 Dome 相互印证 = 通路侧翼佐证。若正式版扩展至 token 级路由需回读
