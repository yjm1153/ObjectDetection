## 核心机制

### 三大 RT-DETR 结构缺陷诊断

| # | 缺陷 | 表现 | DFIR-DETR 应对 |
|---|------|------|---------------|
| 1 | 渐进下采样特征退化 | <32×32 像素物体丢失关键特征 | FIRC3 频域保留高频 |
| 2 | 空间卷积感受野受限 | 固定膨胀比，小目标上下文不足 | DCFA 动态 K-稀疏注意力 |
| 3 | 上采样特征膨胀 | 最近邻插值引入伪影+通道膨胀 | DFPN 振幅归一化上采样 |

### FIRC3 — 频域迭代优化特征聚合

**核心洞察**：将特征聚合重新定义为频域约束最小二乘问题。

**双路架构**：
```
Y = W₃(M(W₁∗X) + W₂∗X)
```
其中 M(·) 为 n 层级联 FIRC 操作。

**频域中间变量**：
```
F_R = F[K]∗ ⊙ F[U_s(F)] + F[ε_b · I_s(F)]
```
- K：深度可分离频域卷积核（softmax 初始化）
- ε_b = σ(b-9.0) + ε：自适应正则化项（ε=10⁻⁵）
- ⊙：逐元素乘积

**自适应权重校正**：
```
W_inv = Avg_s(F[K] ⊙ F_R) / (Avg_s(|F[K]|²) + ε_b)
```
Avg_s(·) 将频谱重排为 s×s 块取平均 → **频域选择性聚合**

**迭代逆求解**：
```
F̂_out = F⁻¹[(F_R − F[K]∗ ⊙ Repeat_s(W_inv)) / ε_b]
```
通过频域共轭操作实现反向传播，抑制低频冗余同时保留高频细节。

### DCFA — 动态跨阶段特征聚合

**DKSA（动态 K-稀疏注意力）**：
```
K = ⌊N · σ(AvgPool(ψ(X)))⌋
```
- ψ：两层卷积门控网络
- 复杂度从 O(N²) → O(NK)
- 理论依据：查询-键互信息服从幂律分布 → K-稀疏近似误差指数级衰减

**SGLU（空间门控线性单元）**：
```
SGLU(X) = X + φ_out(D((φ_act(φ_dw(X_g))) ⊙ X_v))
```
3×3 深度卷积编码局部空间上下文 → 门控决策整合邻域模式。

### DFPN — 动态特征金字塔

**ANUP（振幅归一化上采样）**：
```
F_i↑ = β · U(F_i) = (1/s²) · U(F_i)
```
理论："空间插值使 L1 范数膨胀 s² 倍" → 归一化后 \|F_i↑\|₁ ≈ \|F_i\|₁

**DPSC（双路 Shuffle 卷积）**：标准卷积（语义）+ 深度可分离卷积（细节）→ Channel Shuffle 跨路信息交换。
